<task type="auto">
  <name>Task 2: Implement SyslogTransport class with Unix socket and UDP</name>
  <files>lib/logger-syslog.js</files>
  <action>
Add to lib/logger-syslog.js:

1. SyslogTransport class:
   ```javascript
   class SyslogTransport {
     constructor(config = {}) {
       this.mode = config.mode || 'unix';
       this.facility = config.facility || FACILITY.LOCAL0;
       this.appName = config.appName || 'gsd';
       // Platform-aware socket path
       this.path = config.path || (process.platform === 'darwin' ? '/var/run/syslog' : '/dev/log');
       this.host = config.host || '127.0.0.1';
       this.port = config.port || 514;
     }
   }
   ```

2. _sendUnix(message) method:
   - Create connection via net.createConnection({ path: this.path })
   - On 'error': destroy socket silently (never throw)
   - On 'connect': write message + '\n', then end socket
   - Wrap entire operation in try/catch for safety

3. _sendUdp(message) method:
   - Create socket via dgram.createSocket('udp4')
   - Convert message to Buffer
   - Send to host:port
   - Close socket in callback
   - Wrap in try/catch, never throw

4. send(severity, message, structuredData) method:
   - If severity is null, return immediately (level OFF)
   - Format message using formatMessage()
   - Call _sendUnix or _sendUdp based on this.mode
   - Wrap in try/catch, never throw

Use: require('node:net'), require('node:dgram'), require('node:os')
  </action>
  <verify>
Run: node -e "const s = require('./lib/logger-syslog.js'); const t = new s.SyslogTransport(); t.send(6, 'test from node', {test: 'value'}); console.log('sent')"
Then: journalctl -t gsd --since "1 minute ago" (on Linux)
Expected: Message appears in journal (or silent failure if syslog unavailable - that's OK)
  </verify>
  <done>SyslogTransport sends messages, handles both Unix and UDP modes, never throws</done>
