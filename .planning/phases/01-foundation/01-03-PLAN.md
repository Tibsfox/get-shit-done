     constructor(name = 'unnamed', options = {}) {
       this.name = name;

       this.stopped = false;

   - Return just the milliseconds as Number

4. stop() method:
   - Record endTime = process.hrtime.bigint()
   - Set stopped = true
   - Return elapsed() result

5. exceedsThreshold() method:
   - Return false if no threshold set
   - Return true if elapsedMs() > threshold
   - Useful for warning detection

6. createTimer(name, options) factory function:

   - Provides functional API alternative to class

No external dependencies - uses only process.hrtime.bigint()
